# logos/ — PureVPN logo

Drop the PureVPN logo here.

- **Best:** `.svg` (stays sharp at any size). In Figma: select the logo →
  right-click → **Copy as SVG**, paste into a text file named `logo.svg`;
  OR select it → Export → **SVG**.
- **Okay:** `.png` at 2x or 3x (transparent background).

Helpful variants if you have them:
- `logo-full.svg`  → icon + "purevpn" wordmark (as in the app sidebar)
- `logo-mark.svg`  → just the shield/triangle icon
- `logo-white.svg` → white version for dark backgrounds (the hero uses this)
