# icons/ — icons from the design

Drop any icons here as **`.svg`** (preferred).

You often DON'T need to export these — Claude can recreate most common icons
(search, bell, gear, shield, arrows, sidebar icons, social icons) from a free
icon library that matches the design.

**Only export an icon here if it's custom/unusual** and not a standard icon.
In Figma: select the icon → Export → **SVG**.
